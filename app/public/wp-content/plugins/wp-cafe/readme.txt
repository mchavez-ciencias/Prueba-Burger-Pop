== Description ==
WP Cafe is a one in all solution to manage Online Food Ordering, Restaurant Menu, Restaurant Reservation and more. The online delivery system working with WooCommerce. WPCafe supports both WordPress Shortcode and Elementor Builder. The plugin comes with Restaurant Reservation and Restaurant Menu Features.
